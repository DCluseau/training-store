## But de l'application

L'application est une simulation de boutique de formations. Un utilisateur peut consulter une liste de formations.

## Options disponibles

- Afficher la liste des formations avec son nom, sa catégorie (présentiel ou distanciel), son prix et sa description
- Afficher les catégories par catégorie
- Rechercher une formation par mots-clés

## Manuel d'utilisation

Lors du lancement de l'application, le menu principal s'affiche et une liste d'options s'affiche. Pour sélectionner une option, l'utilisateur doit entrer le numéro de l'option et appuyer sur entrée.
- Option 1 : la liste complète des formations s'affiche puis le menu principal s'affiche à nouveau;
- Option 2 : l'application affiche les catégories disponibles et demande à l'utilisateur de sélectionner une catégorie à afficher. Ensuite, la liste des formations de cette catégorie s'affiche. Enfin, le menu principal s'affiche à nouveau ;
- - Option 3 : l'application demande à l'utilisateur d'entrer des mots-clés, séparés par un +. L'application liste ensuite les formations correspondant aux mots-clés saisis. Enfin, elle retourne au menu principal;
  - Option 4 : l'application se termine.
  Note : si un numéro non présent dans les menus est saisis, le menu principal est à nouveau affiché. Si un caractère invalide (chaîne de caractère) est saisi, l'application affiche une erreur et se termine.
